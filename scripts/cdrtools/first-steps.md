# First steps in CDRTools

[Cdrtools](https://codeberg.org/schilytools/schilytools) is a set of open source software tools for the creation and management of compact discs (CD) and similar optical discs. It was originally created by Jörg Schilling and is used in operating systems based on UNIX, such as Linux and BSD.

CDRTools tools include programs such as "CDRed", which is used to record data on optical discs, such as CD-R, CD-RW and DVD. It also includes `MKISOFS`, which allows you to create images of ISO discs from files and directories in the system, and `READCD`, which is used to read optical discs data and extract them in the system.

## Basic commands

Application version.

```command
cdrtools --version
```

Application help.

```command
cdrtools --help
```

Application information.

```command
info cdrtools
```

Application path.

```command
which cdrtools
```

Application description.

```command
whatis cdrtools
```
