# First steps in Google Chrome

[Google Chrome](https://www.google.com/intl/es_es/chrome/) is a web browser developed by Google. It is one of the most popular and widely used browsers worldwide.

## Basic commands

Application version.

```command
google-chrome --version
```

Application help.

```command
google-chrome --help
```

Application information.

```command
info google-chrome
```

Application path.

```command
which google-chrome
```

Application description.

```command
whatis google-chrome
```
