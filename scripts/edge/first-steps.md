# First steps in Microsoft Edge

[Microsoft Edge](https://www.microsoft.com/es-es/edge) is a web browser developed by Microsoft. It was released in 2015 as the Internet Explorer successor and is designed to be faster, safe and compatible with modern web standards.

EDGE is based on Chromium, the same open source project that drives Google Chrome browser. This transition allowed Microsoft to improve compatibility with existing web pages and provide a stronger navigation experience.

## Basic commands

Application version.

```command
microsoft-edge --version
```

Application help.

```command
microsoft-edge --help
```

Application information.

```command
info microsoft-edge
```

Application path.

```command
which microsoft-edge
```

Application description.

```command
whatis microsoft-edge
```
