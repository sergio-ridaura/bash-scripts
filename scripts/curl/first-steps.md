# First steps in cURL

[Curl](https://curl.se/) is a command line tool used to make applications and receive answers from or to servers using several network protocols, such as HTTP, HTTPS, FTP, SMTP, among others. The name "Curl" is an abbreviation for "Client for URLs" (customer for URL).

Curl is very versatile and offers a wide range of functionalities. It allows GET, Post, Put, Delete applications, and other HTTP actions. You can send and receive data, manage cookies, establish custom headers, follow redirections, and file file downloads from the command line, among other capabilities.

## Basic commands

Application version.

```command
curl --version
```

Application help.

```command
curl --help
```

Application information.

```command
info curl
```

Application path.

```command
which curl
```

Application description.

```command
whatis curl
```
