# First steps in OBS Studio

[OpenSSH](https://www.openssh.com/) is a free and open source implementation of the Secure Shell (SSH) protocol. SSH is a secure networking protocol that allows remote access to computer systems and secure data transfer. OpenSSH provides tools and services to establish encrypted and authenticated connections between computers on a network.

## Basic commands

Application version.

```command
obs-studio --version
```

Application help.

```command
obs-studio --help
```

Application information.

```command
info obs-studio
```

Application path.

```command
which obs-studio
```

Application description.

```command
whatis obs-studio
```
