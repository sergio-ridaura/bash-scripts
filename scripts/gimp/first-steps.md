# First steps in Gimp

[GIMP](https://www.gimp.org/) It is a program dedicated to photographic retouching and graphics creation.

## Basic commands

Application version.

```command
gimp --version
```

Application help.

```command
gimp --help
```

Application information.

```command
info gimp
```

Application path.

```command
which gimp
```

Application description.

```command
whatis gimp
```
