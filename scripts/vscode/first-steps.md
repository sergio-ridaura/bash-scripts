# First steps in Visual Studio Code

[Visual Studio Code](https://code.visualstudio.com/) is an integrated development environment (IDE) that is mainly used to program and edit code. It is developed by Microsoft and is a very popular tool among software developers due to their flexibility, efficiency and wide range of characteristics.

## Basic commands

Application version.

```command
code --version
```

Application help.

```command
code --help
```

Application information.

```command
info code
```

Application path.

```command
which code
```

Application description.

```command
whatis code
```
