# First steps in Blender

[Blender](https://www.blender.org/) is an open source software program used for the creation of 3D graphics, animations, visual effects, modeling, rendering and more. It is a versatile and powerful tool that has gained popularity in the film industry, video games, animation and design.

## Basic commands

Application version.

```command
blender --version
```

Application help.

```command
blender --help
```

Application information.

```command
info blender
```

Application path.

```command
which blender
```

Application description.

```command
whatis blender
```
