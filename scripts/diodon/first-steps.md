# First steps in Diodon

[Diodon](https://launchpad.net/diodon) is a clipboard (clipboard) open source for the Linux desktop environment. A clipboard is a functionality that allows you to copy and paste content, such as text or files, between different applications or windows in the operating system.

## Basic commands

Application version.

```command
diodon --version
```

Application help.

```command
diodon --help
```

Application information.

```command
info diodon
```

Application path.

```command
which diodon
```

Application description.

```command
whatis diodon
```
