# First steps in Inkscape

[Inkscape](https://inkscape.org/) is a program dedicated to the creation and editing of vector graphics.

## Basic commands

Application version.

```command
inkscape --version
```

Application help.

```command
inkscape --help
```

Application information.

```command
info inkscape
```

Application path.

```command
which inkscape
```

Application description.

```command
whatis inkscape
```
