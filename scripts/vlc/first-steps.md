# First steps in VLC

[VLC](https://www.videolan.org/vlc/) is an open source, cross-platform media player. The full name of VLC is VLC media player and it is developed by the non-profit organization VideoLAN. VLC has become one of the most popular and widely used media players due to its ability to play a wide variety of audio and video formats, as well as its availability on different operating systems.

## Basic commands

Application version.

```command
vlc --version
```

Application help.

```command
vlc --help
```

Application information.

```command
info vlc
```

Application path.

```command
which vlc
```

Application description.

```command
whatis vlc
```
