# First steps in qBittorrent

[qBittorrent](https://www.qbittorrent.org/) is an open source, cross-platform BitTorrent client. BitTorrent is a peer-to-peer (P2P) file transfer protocol used to download and share large files efficiently. qBittorrent provides a graphical user interface (GUI) that allows users to easily search, download and manage torrents.

## Basic commands

Application version.

```command
qbittorrent --version
```

Application help.

```command
qbittorrent --help
```

Application information.

```command
info qbittorrent
```

Application path.

```command
which qbittorrent
```

Application description.

```command
whatis qbittorrent
```
