# First steps in Fail2ban

[Fail2ban](http://www.fail2ban.org/) application helps the prevention of intruders in the system, penalizing or blocking remote connection that attempt by brute force access.

## Basic commands

Application version.

```command
fail2ban --version
```

Application help.

```command
fail2ban --help
```

Application information.

```command
info fail2ban
```

Application path.

```command
which fail2ban
```

Application description.

```command
whatis fail2ban
```
